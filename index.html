Accounting App

Launch

Run a local static server from the project folder (Node.js required), then open the app in your browser:

  • Windows: launch.bat
  • macOS: launch.command
  • Linux: ./launch.sh

URL: http://localhost:10800

Install Node.js from nodejs.org if needed.

Before first setup

Create two Google Drive folders: a primary folder and a mirror folder. Share both with your operator Google account using the Editor role. The setup wizard will ask for the share links to these folders and the operator email.
